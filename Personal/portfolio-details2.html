Real Time COVID-19 Database

The Real Time COVID-19 Database is a Python-based data visualization project that retrieves real-time COVID-19 statistics from an API and graphs them using Matplotlib and Pandas. The data is stored in a SQL database and can be customized and modified to display various COVID-19 metrics, such as deaths, cases, and vaccinations. This project won a hackathon, demonstrating its technical excellence and innovation. It is a powerful tool for tracking and analyzing COVID-19 data in real-time.
